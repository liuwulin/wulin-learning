# springboot-rocketmq-demo

#### 前提条件
JDK 1.8 and above
Maven 3.0 and above

#### 功能特性：
功能特性：
- [x] 同步发送
- [x] 同步顺序发送
- [ ] 同步批量发送
- [x] 异步发送
- [x] 异步顺序发送
- [x] 顺序消费
- [ ] 并发消费（广播/集群）
- [x] one-way方式发送
- [x] 事务方式发送
- [ ] 消息轨迹
- [ ] ACL
- [ ] pull消费

#### rocketmq源码学习网址

 - 官网：https://rocketmq.apache.org/ 文档在Documentation
 - github rocketmq整合boot 源码：https://github.com/apache/rocketmq-spring
 - github rocketm源码：https://github.com/apache/rocketmq
